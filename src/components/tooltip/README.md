### tooltip 文字提示

|  参数  | 说明    |  类型   |  可选值  |  默认值  |
|--------|--------|--------|--------|--------|
| placeholder | 输入框占位文本 |   String   | - | - |
| value / v-model | 绑定值 |   String   | - | - |
| disabled | 是否禁用 |   Boolean   | - | false |
| clearable | 是否可清空 |   Boolean   | - | false |
| readonly | 是否只读 |   Boolean   | - | false |
