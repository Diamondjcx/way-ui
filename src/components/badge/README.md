## Badge 标记

#### Attributes

|  参数  |   说明    |  类型   |  默认值  | 可选值 |
|-------|--------|--------|--------|--------|
|  value    | 数值     |  String、Number   |  -  | - |
|  max | 最大值 | Number | - | - |
|  color | 颜色 | String | - | - |
|  visible | 是否显示 | Boolean | true | - |
|  dot | 圆点 | Boolean | false | - |
